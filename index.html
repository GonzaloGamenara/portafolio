Mis Proyectos mas importantes

  • StellarSights

    Un proyecto interactivo donde puedes navegar entre sistemas solares y galaxias, usando datos reales de la NASA. Creado con HTML, CSS y JavaScript, consume la API de la NASA para mostrar información detallada y visualizaciones de planetas y cuerpos celestes, todo sin recargar la página. Una experiencia dinámica y educativa para explorar el universo.

  • Gamelitics

    En este proyecto, desarrollé una base de datos propia en SQLite, donde almacené y limpié información de más de 350,000 videojuegos extraídos de la API RAWG (1990-2025). Utilicé herramientas como Pandas, Matplotlib y Seaborn para el análisis y visualización de datos, y apliqué Machine Learning para predecir el éxito de futuros títulos mediante la identificación de tendencias en la industria.
